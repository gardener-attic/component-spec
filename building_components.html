Building Components

The CNUDIE Component Descriptor model takes no assumption as to how component versions are created. Typically, they are generated during a build process, as this is where their artifacts are generated. About those build processes, the CNUDIE Component Descriptor model also does not take any assumptions.

To make generating Component Descriptors more convenient, there is a re-usable toolset that can be used (https://github.com/gardener/component-cli).
